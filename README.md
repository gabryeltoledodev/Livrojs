# 📘 Lógica de Programação e Algoritmos com JavaScript

## 📌 Status: Em andamento 🚧

Repositório criado para documentar minha evolução em lógica de programação utilizando JavaScript, com foco na construção de uma base sólida para desenvolvimento backend e engenharia de software.

---

## 🎯 Objetivo

Desenvolver raciocínio lógico, domínio das estruturas fundamentais da programação e boas práticas de organização de código.

---

## 🧠 Conceitos praticados

- Variáveis e Tipos de Dados
- Operadores Aritméticos e Lógicos
- Estruturas Condicionais
- Estruturas de Repetição
- Manipulação de Strings
- Entrada e saída de dados com Node.js

---

## 🛠 Tecnologias utilizadas

- JavaScript (ES6+)
- Node.js
- Git
- GitHub
